# Google 試算表同步設定

1. 建立 Google 試算表，例如「總務工作管理」。
2. 在試算表選「擴充功能」→「Apps Script」，以 `Code.gs` 全部內容取代預設程式並儲存。
3. 選擇函式 `install` 並執行一次，完成授權。它會建立 `任務主檔`、`工作紀錄`、`同步紀錄`，並在「執行記錄」顯示 `SYNC_KEY`。
4. 「部署」→「新增部署作業」→「網頁應用程式」。執行身分選「我」；存取權依個人帳號可用選項設定為可讓 GitHub Pages 呼叫。複製 `/exec` 網址。
5. 在 HTML 工具按「設定試算表同步」，輸入網址與 `SYNC_KEY`；密鑰只存於該瀏覽器。
6. 按「同步到 Google 試算表」。每次同步會覆寫 `任務主檔`、`工作紀錄` 的目前快照，並在 `同步紀錄` 留下結果。

## 安全

- 不要將 `SYNC_KEY`、Web App URL、JSON 備份或含個資 CSV 推送至公開 GitHub。
- 同步端沒有提供公開讀取功能；網址加上同步密鑰仍應視為敏感資訊。
